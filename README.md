# CrimeTemplateD

## 🚀 Description
CrimeTemplateD is a backend web application built with Node.js, Express, and MongoDB.  
It allows users to create, manage, and view crime reports (signalements) with authentication and database integration.

## 🌐 Live Demo
👉 https://crimetemplated.onrender.com

## 🛠️ Technologies
- Node.js
- Express.js
- MongoDB Atlas (Mongoose)
- EJS (Embedded JavaScript Templates)
- Express Session

## ✨ Features
- User authentication (login / register)
- Create and manage crime reports
- MongoDB Atlas integration (cloud database)
- Dynamic rendering with EJS
- RESTful routing structure

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Amir-Malakshah/CrimeTemplateD.git
cd CrimeTemplateD

```

Install dependencies:

```bash
npm install

```

🔐 Environment Variables

Create a .env file in the root directory and add:

```env
MONGODB_URI=your_mongodb_atlas_connection_string
SESSION_SECRET=your_secret_key
PORT=5000

```

▶️ Run the Application

Start the server:

```bash
npm start

```

Then open:

```bash
http://localhost:5000
```

Note: You need your own MongoDB Atlas connection string to run this project locally.
