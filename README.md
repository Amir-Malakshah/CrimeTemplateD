# CrimeTemplateD

## 🚀 Description
CrimeTemplateD is a backend web application built with Node.js, Express, and MongoDB.  
It allows users to create, manage, and view crime reports (signalements) with authentication and database integration.

## 🌐 Live Demo
👉 https://crimetemplated.onrender.com

## 🛠️ Technologies
- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS (Embedded JavaScript Templates)
- Express Session

## ✨ Features
- User authentication (login / register)
- Create and manage crime reports
- MongoDB Atlas integration (cloud database)
- Dynamic rendering with EJS
- RESTful routing structure
